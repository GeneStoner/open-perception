**The direction-of-motion input the model receives.** With dot identity stripped away, each panel shows only the motion directions present over time — clockwise (**CW**), translation (**TRANS**, blue, the excitatory drive), counter-clockwise (**CCW**) — for the same four conditions (A–D) as the trajectories above, on the same time axis, so each input panel sits directly beneath its trajectory. **Bar thickness encodes intensity:** rotations run at full strength while the translation is drawn half as thick, because only ~50% of the dots translate coherently. A rotation channel briefly drops out (gaps) while its field translates, and the **TRANS** channel pulses for 40 ms. The key point: the motion swap adds no new directional information. **A and B are identical, as are C and D** — each swap trial reproduces the input of the *opposite* cue's no-swap trial (the swap merely relabels which dots translate, flipping cued ↔ uncued, while the directional input is unchanged). Because the model sees only this input, it cannot distinguish A from B or C from D, and is therefore forced to predict that the cued advantage *reverses* under a motion swap — the prediction the behavioral data refute.
